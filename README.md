# RRT Algorithm
My implementation of the RRT Algorithm on SFML using C++.


## Dependencies
1. SFML 3.0
2. YAML-CPP


## Downloading, Compiling and Running
```bash
git clone https://github.com/mukundbala/RRTAlgorithm.git
cd RRTAlgorithm
mkdir build

chmod +x compile.sh run.sh
./compile.sh

./run.sh
```
